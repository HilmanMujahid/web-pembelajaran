Materi Percabangan

1. Pengertian Percabangan

Percabangan adalah struktur logika dalam algoritma yang digunakan untuk mengambil keputusan berdasarkan suatu kondisi.

Dengan percabangan, program dapat memilih satu aksi tertentu dari beberapa kemungkinan, tergantung pada apakah kondisi yang diuji bernilai benar (true) atau salah (false).

Percabangan sangat penting karena membuat program menjadi:

  • Lebih dinamis
  • Dapat merespons situasi yang berbeda
  • Menyerupai cara manusia mengambil keputusan

2. Tujuan Penggunaan Percabangan

Tujuan utama percabangan dalam algoritma adalah:

  • Mengontrol alur jalannya program
  • Menentukan keputusan berdasarkan kondisi tertentu
  • Membuat program lebih fleksibel dan efisien
  • Menyesuaikan output dengan input yang diberikan

3. Konsep Dasar Percabangan

Percabangan bekerja dengan pola IF ‚Äì THEN ‚Äì ELSE, yaitu:

  • IF (jika) ‚Üí kondisi diuji
  • THEN (maka) ‚Üí perintah dijalankan jika kondisi benar
  • ELSE (jika tidak) ‚Üí perintah dijalankan jika kondisi salah

Secara sederhana: Jika kondisi terpenuhi, lakukan aksi A. Jika tidak, lakukan aksi B.

4. Contoh Kasus dalam Kehidupan Sehari-hari

Contoh sederhana percabangan dapat ditemukan pada lampu lalu lintas:

Warna Lampu Keputusan
Merah Berhenti
Kuning Pelan-pelan
Hijau Jalan

Dari contoh ini:

  • Input ‚Üí Warna lampu
  • Proses ‚Üí Mengecek kondisi warna
  • Output ‚Üí Keputusan (berhenti, pelan, jalan)

5. Identifikasi Input dan Output

Dalam sebuah algoritma percabangan, penting untuk mengenali:

Input
Data yang masuk dan akan diuji (contoh: warna lampu)

Output
Hasil keputusan yang diambil oleh program (contoh: berhenti, pelan, melaju)

6. Contoh Pseudocode Percabangan

Berikut contoh pseudocode berdasarkan kasus lampu lalu lintas:

INPUT warna_lampu

IF warna_lampu = "merah" THEN
    OUTPUT "Berhenti"
ELSE IF warna_lampu = "kuning" THEN
    OUTPUT "Pelan-pelan"
ELSE IF warna_lampu = "hijau" THEN
    OUTPUT "Jalan"
ELSE
    OUTPUT "Warna tidak dikenali"
END IF

Pseudocode ini menunjukkan bagaimana komputer:

  • Membaca input
  • Mengecek kondisi
  • Mengambil keputusan yang sesuai

7. Jenis-Jenis Percabangan

  • Percabangan Tunggal (IF)
    Digunakan jika hanya ada satu kondisi.
  • Percabangan Dua Arah (IF ‚Äì ELSE)
    Digunakan jika ada dua kemungkinan keputusan.
  • Percabangan Banyak Pilihan (IF ‚Äì ELSE IF ‚Äì ELSE)
    Digunakan jika terdapat lebih dari dua kondisi.

8. Pentingnya Percabangan dalam Pemrograman

Percabangan digunakan hampir di semua program, seperti:

  • Login dan autentikasi
  • Game (menang, kalah, lanjut)
  • Sistem nilai (lulus atau tidak)
  • Lampu lalu lintas dan sistem otomatis

Tanpa percabangan, program hanya akan berjalan lurus tanpa keputusan.
